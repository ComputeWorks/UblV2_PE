# UBL v2 - SUNAT (.Net Standard 2.0)
Estandar UBL v2 que definie los documentos de la Facturacion Electronica SUNAT

## Nuget 
Install: 
```bash
dotnet add package Ubl.V2.Pe
```  
[Ubl.V2.Pe](https://www.nuget.org/packages/Ubl.V2.Pe)
